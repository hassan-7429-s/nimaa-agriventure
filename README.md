# FoodVentures Website Clone (Static Project)

A lightweight static implementation of FoodVentures' content structure designed from https://foodventures.eu/#about.

## Structure
- `index.html` : Single-page website with navigation and homepage sections
- `styles.css` : Core styling
- `script.js` : Mobile nav toggle behaviour

## Run
- Open `index.html` in a browser.
- Or use npm to run the local dev server:
  - `npm install`
  - `npm run dev`
- Optionally run a local web server:
  - Python 3: `python -m http.server 8000`
  - Node.js: `npx serve .`

## Features
- Hero section, About, Technology, Operations, Investors, Sustainability, Join Us, Contact
- Responsive header and mobile menu
- Clean modern layout inspired by FoodVentures branding.
